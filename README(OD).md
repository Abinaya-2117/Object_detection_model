# Tesla Autopilot Clone – Object Detection

## Project Overview

This project implements an **object detection system inspired by Tesla Autopilot**, capable of detecting objects such as vehicles, pedestrians, and other road elements from images.

The system is built using the **YOLOv8 deep learning model** and trained on the **KITTI dataset**, a widely used dataset for autonomous driving research.

The objective of this project is to simulate a simplified perception module used in autonomous driving systems, where the model detects and localizes objects in real-world driving scenes.

---

# Dataset

The project uses the **KITTI Object Detection Dataset**, which contains labeled images captured from a car-mounted camera.

### Dataset Features

* Real-world driving scenes
* Annotated bounding boxes
* Multiple object classes such as:

  * Car
  * Pedestrian
  * Cyclist
  * Truck
  * Van
  * Tram

### Dataset Format

The dataset was converted into **YOLO format**, where each label file contains:

```
<class_id> <x_center> <y_center> <width> <height>
```

All coordinates are normalized between **0 and 1**.

---

# Project Workflow

## 1. Data Preprocessing

The dataset was prepared by:

* Organizing images and labels into YOLO format
* Splitting the dataset into:

  * Training set
  * Validation set
* Creating a `data.yaml` configuration file containing:

  * Dataset paths
  * Class names
  * Number of classes

---

## 2. Model Selection

This project uses **YOLOv8 (You Only Look Once Version 8)** for object detection.

### Why YOLOv8?

* Real-time object detection
* High accuracy
* Lightweight and efficient
* Easy training pipeline

The **YOLOv8 Nano (yolov8n)** model was used to keep training efficient and suitable for Kaggle GPU environments.

---

## 3. Model Training

The model was trained using the **Ultralytics YOLOv8 framework**.

Example training command:

```python
from ultralytics import YOLO

model = YOLO("yolov8n.pt")

model.train(
    data="data.yaml",
    epochs=10,
    imgsz=640,
    name="yolov8n-kitti"
)
```

Training outputs were saved in:

```
runs/detect/yolov8n-kitti/train
```

# Running the Project

## Train the Model

```python

model = YOLO("yolov8n.pt")

```

---

## Run Object Detection on an Image

```python

model = YOLO("runs/detect/yolov8n-kitti/train/weights/best.pt")

results = model("test_image.jpg", show=True)
```
